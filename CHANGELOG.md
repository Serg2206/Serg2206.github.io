# Changelog / История изменений

Все заметные изменения в проекте будут документированы в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и этот проект придерживается [Semantic Versioning](https://semver.org/lang/ru/).

## [Unreleased] - В разработке

### Добавлено / Added
- Шаблоны для Issues (Bug Report, Feature Request)
- Шаблон для Pull Request
- Руководство по внесению вклада (CONTRIBUTING.md)
- Кодекс поведения (CODE_OF_CONDUCT.md)
- Политика безопасности (SECURITY.md)
- GitHub Actions workflows:
  - Автоматический деплой на GitHub Pages
  - Проверка битых ссылок
  - Проверка качества кода

- Dependabot для автоматического обновления зависимостей

## [1.0.0] - 2025-01-15

### Добавлено / Added
- Начальная версия научно-медицинской платформы SSV Prof
- Научно-медицинская библиотека
- Информация о программах стажировки
- Дистанционное консультирование
- Адаптивный дизайн для мобильных устройств
- MIT License

### Функциональность / Features
- Интерактивная библиотека публикаций
- Формы для записи на стажировки и консультации
- Responsive дизайн
- Мультиязычная поддержка (RU/EN)

## Типы изменений / Types of Changes

- **Добавлено / Added** - новая функциональность
- **Изменено / Changed** - изменения в существующей функциональности
- **Устарело / Deprecated** - функциональность, которая будет удалена
- **Удалено / Removed** - удалённая функциональность
- **Исправлено / Fixed** - исправления ошибок
- **Безопасность / Security** - исправления уязвимостей
