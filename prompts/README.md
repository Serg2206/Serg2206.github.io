# Пакет промптов: ИИ-ассистент для фриланса и подработок

Цена: 499 руб.
Формат: PDF + исходный Markdown

---

## Описание

15 готовых промптов для ChatGPT / Claude / Gemini, чтобы работать быстрее:
- писать тексты и посты
- общаться с заказчиками
- делать ТЗ и оценку задач
- генерировать идеи и структуры

Промпты проверены на практике — копируешь, вставляешь, получаешь результат.

---

## Как использовать

1. Скопируй промпт
2. Вставь в СhatGPT / Claude / Gemini
3. Замени текст в фигурных скобках на свои данные
4. Получи готовый результат

## Промпты

### 1. Переписка с заказчиком — вежливый отказ от проекта

```
Ты — профессиональный фрилансер. Напиши вежливый, но уверенный ответ заказчику, в котором отказываешься от проекта по причине перегруза работой. 

Укажи, что не сможешь уложиться в срок, и порекомендуй обратиться к другим специалистам. Сохрани тон дружелюбным. Не обвиняй заказчика.

Контекст:
- Ниша: {копирайтинг / дизайн / разработка / другие услуги}
- Срок, который предлагает заказчик: {указать срок}
- Твоя текущая загрузка: {полная / ограниченная}

```

### 2. Перевод ТЗ в пошаговый план с дедлайнами

```
Я — фрилансер, получил ТЗ от заказчика. Разбей его на конкретные шаги с примерами, что именно нужно сделать на каждом этапе, и укажи реалистичные сроки (в рабочих днях).

ТЗ:
{вставь ТЗ сюда}

Формат ответа:
1. Шаг 1 — срок
2. Шаг 2 — срок
...

```

### 3. Создание технического задания под заказчика

```
Напиши подробное ТЗ на {название услуги / продукта} для заказчика, чтобы он мог его передать исполнителю.

Укажи:
- цели
- требования
- формат сдачи
- критерии приёмки

Контекст: {опиши задачу своими словами}

```

### 4. Генерация идей для постов в соцсетях

```
Предложи 10 идей постов для {Instagram / Telegram / VK / LinkedIn} для {ниши: дизайн / подработка / обучение / товары}.

Правила:
- 3 поста с пользой (лайфхаки, советы)
- 3 поста с личным опытом
- 2 вовлекающих вопроса
- 2 продающих (без спама)

Пример формата: тезис + короткий текст

```

### 5. Описание услуги для биржи фриланса / сайта

```
Напиши продающее, но честное описание услуги {копирайтинг / дизайн логотипа / настройка рекламы / разработка сайта} для размещения на фриланс-биржах или в соцсетях.

Структура:
1. Кто я и что предлагаю
2. Как я работаю (этапы)
3. Что получает заказчик
4. Примеры работ / портфолио (оставь плейсхолдеры)

Мой опыт: {опыт в годах / проектах / стеке}

```

### 6. Оценка стоимости проекта

```
Я — фрилансер в нише {ниша}. Рассчитай примерную стоимость проекта:
{опиши проект}

Учитывай:
- мою ставку в час / за проект
- сложность
- сроки
- риски

Выведи: общая цена, срок, что включено, что оплачивается отдельно.

```

### 7. Создание коммерческого предложения (КП)

```
Напиши коммерческое предложение на услугу {название услуги} для компании {B2B / B2C}.

Структура:
- вступление (1 абзац)
- pains клиента (2–3 пункта)
- моё решение (с примерами)
- цены и сроки
- преимущества и гарантии
- призыв к действию

Данные:
{вставь релевантную информацию}

```

### 8. Рерайтинг текста под продажу

```
Перепиши текст так, чтобы он продавал. Сохрани смысл, но добавь:
- выгоды для клиента (не про features, а про результат)
- доказательства
- призыв к действию

Исходный текст:
{вставь текст}

Цель: {продажа товара / услуги / записи на консультацию}

```

### 9. Структура для email-рассылки

```
Придумай структуру email-рассылки для {анонс / приветствие / продажа / напоминание}.

Укажи:
- тема (3 варианта)
- открытие (цепляющее первое предложение)
- основную часть (2–3 пункта)
- призыв к действию
- подпись

Цель письма: {опиши}

```

### 10. Создание портфолио с нуля (текстовая часть)

```
Напиши текстовую часть портфолио для ниши {дизайн / копирайтинг / разработка / маркетинг}.

Мне нужно:
- краткое описание профиля (2–3 предложения)
- 3 примера projects с описанием: задача → решение → результат
- список услуг
- клиенты / отзывы (плейсхолдеры)
- контакты

Данные обо мне: {опыт, специализация}

```

### 11. Ответ на негативный отзыв

```
Напиши профессиональный ответ на негативный отзыв о {товаре / услуге}.

Цель:
- не спорить
- показать, что ты решаешь проблемы
- оставить хорошее впечатление у других читателей

Текст отзыва:
{вставь текст}

Сценарий: {ошибка на нашей стороне / клиент не разобрался / спорная ситуация}

```

### 12. Планирование контента на месяц

```
Сделай контент-план на 4 недели для {Telegram / Instagram / YouTube / блог} в нише {ниша}.

Условия:
- 3 поста в неделю
- соблюдай формулу: 1 польза, 1 развлечение, 1 продажа
- добавь темы и тезисы для каждого поста

Цель: {продажи / подписчики / узнаваемость}

```

### 13. Генерация идей для продуктов

```
Предложи 10 идей цифровых продуктов / мини-услуг, которые можно продавать в нише {ниша}.

Критерии:
- можно реализовать за 1–5 часов
- не требует больших вложений
- потенциальный спрос есть

Для каждой идеи: название, что внутри, кому продавать.

```

### 14. Анализ конкурентов

```
Сделай сравнительный анализ двух конкурентов в нише {ниша}.

Для каждого:
- целевая аудитория
- ценовая политика
- преимущества
- недостатки
- что можно взять на вооружение

Конкуренты:
1. {имя / ссылка}
2. {имя / ссылка}

```

### 15. Скрипт холодного outreach / рассылки

```
Написи короткий, вежливый шаблон сообщения для холодного outreach потенциальным клиентам / партнёрам.

Цель: {запросить сотрудничество / предложить услугу / попросить отзыв}
Ниша: {ниша}
Моя суперсила: {в чём ты силён}

Формат: сообщение на 4–6 предложений, без спама.

```
