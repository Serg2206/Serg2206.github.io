## Prompts

### 1. Campaign ideas from a single brief

```
You are a senior performance marketer. Turn this brief into 6–8 campaign angles. For each angle:
- core message
- primary emotion
- channel fit (Meta, TikTok, YouTube, email, search)
- first CTA

Brief:
{your brief}
```

### 2. Headlines and hooks for Meta / TikTok / YouTube

```
Write 15 short headlines and hooks for {Meta / TikTok / YouTube Shorts / Reels} in the {niche} niche.

Rules:
- 5 curiosity hooks
- 5 pain-point hooks
- 5 benefit-first hooks
- each ≤ 8 words for the first read
- add one suggested visual or format for each

Include 3 variations for a cold audience and 2 for retargeting.
```

### 3. Audience segmentation worksheet

```
I am marketing {product or service}. Create an audience segmentation template with:

Segments (3–6):
- name
- demographics / firmographics
- core pain
- trigger to buy
- preferred channel
- message to test

Add a short targeting suggestion for each segment.
```

### 4. Email welcome sequence (3 emails)

```
Create a 3-email welcome sequence for new subscribers after they join {lead magnet name}.

Goal: build trust and move them toward a paid offer.

Include for each:
- subject line (3 options)
- preview text
- body (short, 150–200 words)
- CTA
```

### 5. Sales page copy that converts

```
Write sales page copy for {product / service}.

Structure:
- headline + subhead
- 3 pain points with short stories
- 3 transformation bullets
- social proof placeholders
- offer stack + bonuses
- risk reversal
- 2 CTAs (one soft, one hard)

Tone: {authoritative / friendly / playful}
```

### 6. Competitive ad analysis

```
Analyze these 3 current ads from competitors and extract patterns that work and don’t work.

For each ad:
- hook style
- proof element
- offer structure
- CTA
- your takeaway

Then suggest 3 angles I can test this week to beat them.
```

### 7. Retargeting copy bank

```
Write 8 retargeting snippets for people who:
- visited the site but didn’t buy
- abandoned the cart
- clicked an ad but didn’t convert
- engaged with content but didn’t opt in

Include:
- 3 short email / DM lines
- 3 ad captions
- 2 SMS-style lines
Keep each under 100 characters when possible.
```

### 8. Offer builder

```
Build a new small paid offer for my audience in the {niche} niche.

Constraints:
- price between {price range}
- delivery in under {hours / days}
- easy to explain in one sentence

Output:
- offer name
- one-line promise
- 3 deliverables inside
- bonuses
- guarantee
- launch channel suggestion
```

### 9. Weekly content calendar

```
Create a 4-week content calendar for {Instagram / LinkedIn / X / YouTube / TikTok} for {niche brand}.

Mix:
- 2 value / education
- 2 social proof / case study
- 1 behind-the-scenes
- 1 soft promo

For every piece:
- topic
- hook
- format
- best time to publish
```

### 10. Funnel audit checklist

```
Audit this marketing funnel and flag the 3 biggest leak points.

Funnel description:
{simple description}

For every leak:
- where it happens
- why it hurts
- 2 concrete fixes
```

### 11. Referral / affiliate copy

```
Write referral/affiliate copy for partners to share {product / service}.

Include:
- one-liner for messengers
- 3 LinkedIn / X versions
- 2 email versions
- what NOT to say
```

### 12. Pricing and packaging helper

```
Suggest 3 pricing and packaging options for a {niche} audience with price points:
- entry: {amount}
- core: {amount}
- premium: {amount}

For each tier:
- name
- what’s inside
- who it’s for
- why upgrade from the tier below
```

### 13. Case study structure

```
Write a case study structure for a {client type} win.

Sections:
- challenge
- insight
- action
- results (with placeholders)
- pull quote placeholders

Also give 3 headlines I can test.
```

### 14. Press / media outreach

```
Write a short pitch to {journalist / blogger / podcaster} about {story / release}.

Rules:
- under 180 words
- explain why it fits their audience
- one clear ask
- no fluff
```

### 15. Ad A/B testing planner

```
Design an A/B test plan for {product / campaign}.

Include:
- hypothesis
- variable tested
- audience
- KPI and minimum sample
- success criteria
- how long to run

Suggest 5 strong test ideas.
```
