# Prompt Pack: AI Assistant for Freelancers & Side Gigs

Price: $9–$19 USD (€9–€18)

## Description

15 ready-to-use prompts for ChatGPT / Claude / Gemini to work faster and earn more:
- write texts, posts, and emails
- negotiate with clients
- turn briefs into step-by-step plans with deadlines
- generate ideas and structures

Prompts are tested in real freelance workflows — copy, paste, get a result.

## How to use

1. Copy the prompt
2. Paste into ChatGPT / Claude / Gemini
3. Replace placeholders in curly braces with your data
4. Get a finished result

## Prompts

### 1. Client communication — polite decline

```
You are a professional freelancer. Write a polite but firm reply declining a project due to being at full capacity.

Keep the tone friendly. Do not blame the client. Recommend they find another specialist.

Context:
- Niche: {copywriting / design / development / other services}
- Their deadline: {deadline}
- Your current availability: {fully booked / limited}
```

---

### 2. Turn a brief into a step-by-step plan with deadlines

```
I am a freelancer who just received a client brief. Break it into concrete steps with examples of what to deliver on each stage, and give realistic deadlines in working days.

Brief:
{paste brief here}

Answer format:
1. Step 1 — deadline
2. Step 2 — deadline
...
```

---

### 3. Write a detailed client-ready brief / statement of work

```
Write a detailed brief for {service / product name} so a client can hand it to a contractor.

Include:
- goals
- requirements
- delivery format
- acceptance criteria

Context: {describe the task in plain language}
```

---

### 4. Social media post ideas

```
Suggest 10 post ideas for {Instagram / Telegram / X / LinkedIn / TikTok} in the {design / side gigs / learning / digital products} niche.

Rules:
- 3 value posts (how-tos, tips)
- 3 personal experience posts
- 2 engagement questions
- 2 selling posts (no spam)

Format: hook + short draft.
```

---

### 5. Service description for freelancer platforms / landing page

```
Write a compelling, honest service description for {copywriting / logo design / ads setup / website development} to post on freelancer platforms or social media.

Structure:
1. Who I am and what I offer
2. How I work (steps)
3. What the client gets
4. Work samples / portfolio (placeholders)

My experience: {years / projects / stack}
```

---

### 6. Project price estimation

```
I am a freelancer in {niche}. Estimate the price for this project:
{describe project}

Consider:
- my hourly / per-project rate
- complexity
- timelines
- risks

Output: total price, timeline, what's included, what's billed separately.
```

---

### 7. Write a commercial proposal (sales letter)

```
Write a commercial proposal for {service name} aimed at {B2B / B2C} clients.

Structure:
- opening (1 short paragraph)
- client pains (2–3 bullets)
- my solution with examples
- pricing and timeline
- benefits and guarantees
- call to action

Data:
{relevant info}
```

---

### 8. Rewrite a sales-oriented version of any text

```
Rewrite this text so it sells. Keep the core message, but add:
- client benefits (results, not features)
- proof points
- call to action

Original text:
{original text}

Goal: {sell a product / service / consultation booking}
```

---

### 9. Email newsletter structure

```
Create an email outline for {announcement / welcome / sales / reminder}.

Include:
- 3 subject line options
- opening hook
- main body (2–3 points)
- call to action
- sign-off

Email goal: {describe goal}
```

---

### 10. Build a freelance portfolio (text part)

```
Write the text part of a portfolio for {design / copywriting / development / marketing}.

I need:
- short profile (2–3 sentences)
- 3 project examples: problem → solution → result
- service list
- clients / testimonials (placeholders)
- contacts

About me: {experience, specialization}
```

---

### 11. Respond to a negative review

```
Write a professional response to a negative review about {product / service}.

Goal:
- don't argue
- show you solve problems
- leave a good impression for other readers

Review text:
{review text}

Scenario: {our mistake / customer confusion / gray area}
```

---

### 12. Monthly content plan

```
Create a 4-week content plan for {Telegram / Instagram / YouTube / blog} in the {niche} niche.

Requirements:
- 3 posts per week
- mix: 1 value, 1 entertainment, 1 sales
- topic + short thesis for every post

Goal: {sales / followers / brand awareness}
```

---

### 13. Digital product / micro-service ideas

```
Suggest 10 ideas for digital products / micro-services to sell in the {niche} niche.

Criteria:
- can be built in 1–5 hours
- no big upfront cost
- clear demand

For each: name, contents, target customer.
```

---

### 14. Competitor quick comparison

```
Create a short competitive analysis of two players in the {niche} niche.

For each:
- target audience
- pricing model
- strengths
- weaknesses
- what you can learn from them

Competitors:
1. {name / link}
2. {name / link}
```

---

### 15. Cold outreach / partnership message template

```
Write a short, polite cold outreach message to potential clients / partners.

Goal: {ask for collaboration / offer service / request testimonial}
Niche: {niche}
My superpower: {what you're especially good at}

Format: 4–6 sentences, no spam.
```
